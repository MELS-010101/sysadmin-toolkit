🛠️ SysAdmin-Toolkit

Production-ready CLI toolkit для автоматизации рутинных задач системного администратора

🐧 Linux 🍎 macOS 🪟 Windows Docker 📦 npm / NuGet

Возможности

📊 System Health

Мониторинг CPU, RAM, дискового пространства и состояния сервисов в реальном времени.

📝 Log Cleanup

Автоматическая ротация, архивация и очистка логов с поддержкой S3/MinIO.

🌐 Network Audit

Сканирование сети, проверка firewall, аудит безопасности.

✅ Safe by Default

Dry-run режим и подтверждение всех деструктивных операций.

🐳 Docker Ready

Готовый стек Prometheus + Grafana для мониторинга из коробки.

📦 Easy Install

Установка через npm, NuGet или из исходников за 1 минуту.

Быстрая установка

git clone https://github.com/MELS-010101/sysadmin-toolkit.git cd sysadmin-toolkit chmod +x src/linux/*.sh src/macos/*.sh lib/*.sh make install sat --help
Скачать с GitHub 📖 Документация

Docker развёртывание

cd integrations/docker docker-compose up -d # Сервисы доступны по адресам: # Grafana: http://localhost:3000 (admin/admin) # Prometheus: http://localhost:9090 # MinIO: http://localhost:9001
